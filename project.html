My first Devlog
This is the devlog I uploaded last year in November. I mostly showcased the FPS controller and shooting mechanics. The FPS controller's code was mostly borrowed from Comp-3 Interactive's lengthy FPS controller series. A lot of it will mostly change as the development goes along, and as ideas for the game change.

However, I tried making my own shooting controller. In the beginning it was pretty buggy, the recoil system made bullets go all over the place, the ammo wouldn't be subtracted correctly, there were problems with animations not playing properly, etc. Moving on from working on casual 2D games for Android in a professional environment, to working on a 3D FPS shooter was not an easy transition, but it has certainly been a really good learning experience. I have learned a lot about rigid bodies, raycasting, animation and the 3D environment in general.
